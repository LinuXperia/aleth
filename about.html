Aleth est une application de cartes mémoires créée par Thomas HUET.

Aleth vous permet de créer des cartes pour vous aider à mémoriser tout ce que vous voulez. Quand vous apprenez quelque chose de nouveau, créez une carte et Aleth s'assurera que vous ne l'oubliiez jamais.
Il est conseillé de réviser vos cartes chaque jour (sinon vous en aurez plus à réviser le lendemain). Les révisions ne prennent en général que quelques minutes par jour.
Aleth a été conçu pour être utilisé comme décrit dans cette longue dissertation (en anglais).

Aleth se veut aussi facile à utiliser que possible et a donc un nombre de fonctionnalités restreint. Si vous recherchez une application plus complète qui permette de créer différents jeux de cartes et de les partager, je vous recommande Anki qui dispose d'une large bibliothèque de cartes créées par sa communauté.

L'application est open source (AGPLv3) et le code source est disponible sur GitHub.
